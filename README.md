# Streamers API

## General info

A server application project based on the REST API architecture for creating and managing streamers and collecting positive and negative votes for each of them in order to prepare a ranking.

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
